# ÉLARA

O projeto ÉLARA foi desenvolvido com o objetivo de criar uma aplicação web voltada ao segmento de moda feminina, simulando um e-commerce moderno com foco em experiência do usuário, interatividade e design responsivo.

A proposta do projeto é demonstrar a construção de uma loja virtual funcional, reunindo recursos comuns de plataformas reais de e-commerce em uma aplicação front-end.

## 🎯 Objetivo

O principal objetivo deste projeto foi desenvolver uma aplicação funcional que:

* Simule uma experiência de compra online
* Aplique conceitos de desenvolvimento web na prática
* Explore interface responsiva e usabilidade
* Demonstre funcionalidades comuns de um e-commerce moderno

## 💻 Desenvolvimento

Durante o desenvolvimento, foram aplicados conceitos fundamentais de desenvolvimento web, incluindo:

* Estruturação da aplicação com HTML
* Estilização e responsividade com CSS
* Implementação de interações e funcionalidades com JavaScript

O projeto foi pensado para simular um sistema real, focando na organização visual, navegação e experiência do usuário.

## 🛍️ Funcionalidades

A aplicação conta com recursos como:

* Catálogo de produtos
* Filtros por categorias
* Carrinho de compras
* Checkout simulado
* Login e cadastro
* Rastreamento de pedidos
* Dashboard administrativo
* Estrutura PWA (Progressive Web App)

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
